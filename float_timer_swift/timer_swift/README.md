##  关于如何对抗苹果审核

苹果可能会因为你使用了 `backgroud mode` 而拒绝你，这个时候你可以在APP里添加一个系统播放器，就播放画中画的使用教程（时间越长越好），有了系统播放器，你就可以理所当然的使用画中画了。

或者用一个web，web里的播放器也会调用系统播放器，用这种方式的居多。

总之就是告诉苹果，我需要在后台放视频。
